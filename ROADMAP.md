# 🧭 Todo App Roadmap

A professional-grade Todo App built for resume showcasing and real-life productivity.

---

## ✅ Version 1.0 – MVP (Minimum Viable Product)
> Core functionality with clean UI and local persistence.

### Features
- [ ] Add / Edit / Delete tasks
- [ ] Mark tasks as complete/incomplete
- [ ] Due date & time for tasks
- [ ] Categorize tasks (e.g., Work, Study, Personal)
- [ ] Persist data using `localStorage`

### UI
- [ ] Task cards with checkboxes
- [ ] Color-coded categories
- [ ] Simple date picker

---

## 🚀 Version 1.1 – Productivity Booster
> Add filters, sorting, and improved user control.

### Features
- [ ] Task priority (Low, Medium, High)
- [ ] Search bar
- [ ] Sort & filter (by date, category, priority)
- [ ] Light/Dark mode
- [ ] Basic analytics (Tasks done today/this week)

### UI
- [ ] Filter chips (category, priority)
- [ ] Task counter
- [ ] Toggle sort buttons

---

## 🌐 Version 2.0 – Cloud Sync
> Move to authenticated users and real-time sync.

### Features
- [ ] User authentication (Email + Google)
- [ ] Sync tasks to Firebase Firestore
- [ ] Auto sync across sessions/devices
- [ ] Account settings (update/delete account)

### UI
- [ ] Login / Register pages
- [ ] Settings page
- [ ] Toast notifications & loaders
